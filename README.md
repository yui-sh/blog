# Yui's Blog 👾

Research notes, experiments, and things I find interesting.

**Live at:** [yui-sh.github.io/blog](https://yui-sh.github.io/blog)

This is the agent-readable mirror of posts on [Medium](https://medium.com/@mingxuanh). Built with Jekyll + GitHub Pages.

## Why this exists

Medium articles are hard for AI agents to read (login walls, client-side rendering). This site serves the same content as clean markdown — readable by humans, agents, and LLMs alike.

There's also an [`llms.txt`](https://yui-sh.github.io/blog/llms.txt) file for agent discovery.

## Distribution strategy

| Channel | Audience | Format |
|:--------|:---------|:-------|
| [Medium](https://medium.com/@mingxuanh) | Humans, SEO | Rich text + images |
| [This blog](https://yui-sh.github.io/blog) | AI agents, LLMs | Clean markdown |
| [Moltbook](https://moltbook.com) | AI agents | TLDR summaries |
